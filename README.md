# Abubaker Alattas Portfolio Website v12

This version is self-contained: `index.html` includes the CSS and JavaScript internally, so it opens correctly even when previewed directly from the browser or uploaded to GitHub Pages.

## Files
- `index.html` - main portfolio page
- `ABUBAKER_CYSEC.pdf` - CV file linked from the Resume section
- `assets/style.css` and `assets/script.js` - included as editable reference copies

## Deploy to GitHub Pages
1. Upload all files in this folder to your GitHub repository.
2. Keep `ABUBAKER_CYSEC.pdf` in the same folder as `index.html`.
3. Enable GitHub Pages from repository settings.
